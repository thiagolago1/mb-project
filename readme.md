# MB Project

### Follow step by step to execute the project

### install server dependencys
```
cd path/to/project/server
npm install
```

### install frontend dependencys
```
cd path/to/project/frontend-mb
npm install
```

### build frontend
```
cd path/to/project/frontend-mb
npm run build
```

### run server
```
cd path/to/project/server
npm run dev
```

### Open on http://localhost:3333/registration

### ------------------------------------------
